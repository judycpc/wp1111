## Deployment 流程
- 新增 Dockerfile
- 在前後端的 package.json 加上所需 scripts (production mode & deploy 相關)
- 在 Railway 新增 project，選擇 github 並設定權限
- 選擇 github repo & deploy now，Railway 自動偵測 Dockerfile 並開始 build
- 待 deploy 完成，選擇 generate domain 來取得服務網址
