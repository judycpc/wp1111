## II. 本地端安裝與測試步驟

### 前端
請在 `final/frontend` 資料夾下執行以下指令：  
```
yarn        // 安裝所需套件  
yarn start  // 啟動伺服器
```
注意：本專案所串接的 Imgur API 有阻擋 `localhost` 與 `127.0.0.1` 之情形，建議在瀏覽器開啟 `http://<YOUR_IP_ADDRESS>:3000` 以正常操作與顯示介面。

### 後端
請見 `final/backend/README.md` 說明。
